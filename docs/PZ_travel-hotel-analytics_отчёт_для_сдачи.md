## 3. Реализованный функционал

### 3.1. Базовый MVP (Replit — UI Shell)

- трёхколоночный интерфейс в стиле Bloomberg Terminal;
- русскоязычный UI, тёмная тема AIKIVAVIORA;
- KPI-карточки, карточки отелей, матрица сравнения;
- журнал активности, источники данных, пульс рынка;
- исследовательский чат (демо-диалог).

### 3.2. Доработки в Cursor (фазы 1–3)

| Модуль | Реализация |
|--------|------------|
| **Мой список** | Watchlist в localStorage, вкладка «Мои», кнопка в статус-баре |
| **Сравнение** | До 9 отелей, единое состояние, панель активных сравнений, снимок |
| **Отчёты** | Сохранение из чата + снимок сравнения, экспорт Markdown |
| **Погода MCP** | Weather Panel: сейчас / 7 дней / почасово, график 30 дней |
| **Каталог** | 24 демо-отеля; целевой STR 847 — вкладка «Все (847)» с пометкой «в разработке» |
| **Прозрачность UX** | Бейджи Демо / В разработке / Заглушка / Локально — не выглядит как баг |
| **API stubs** | `GET/PUT /api/workspace`, `GET/POST /api/reports` |
| **DB stubs** | Drizzle schema `workspace`, `reports` (без миграции) |

### 3.3. Weather MCP (задел под ДЗ-2)

| Эндпоинт | MCP tool |
|----------|----------|
| `GET /api/weather/current` | `get_current_conditions` |
| `GET /api/weather/daily` | `get_forecast` (daily) |
| `GET /api/weather/hourly` | `get_forecast` (hourly) |

Документация: `docs/WEATHER_MCP.md`.

## 4. Тестирование (обязательные сценарии)

### Сценарий 1 — Обзор интерфейса и каталога

| Поле | Результат |
|------|-----------|
| **Действие** | Открыть приложение, статус-бар «24 / 847», кнопка каталога |
| **Ожидание** | Вкладка «Все (847)», баннер «в разработке», 24 демо-отеля |
| **Проверка** | Фильтры Мои / 5★ / Курорты работают на демо-наборе |
| **Скрин** | `02_kpi_hotels.png` |

### Сценарий 2 — Погода MCP по активному отелю

| Поле | Результат |
|------|-----------|
| **Действие** | Выбрать отель (напр. Four Seasons, Нью-Йорк), открыть «Погода в регионе» |
| **API** | `localhost:3001/api/weather/*` → Weather MCP stdio |
| **Ожидание** | Текущие условия, прогноз 7 дней, почасовой, график температуры |
| **Скрин** | `03_weather_mcp.png` |

### Сценарий 3 — Сравнение и отчёт

| Поле | Результат |
|------|-----------|
| **Действие** | Добавить 2–3 отеля в сравнение, сохранить снимок; в чате — «Отчёт по рынку» → «Сохранить отчёт» |
| **Ожидание** | Матрица метрик, отчёты в «Исследовательской среде», экспорт `.md` |
| **Хранение** | localStorage (помечено бейджем «Локально») |

## 5. Что в разработке (не баг)

|---------|--------|
| Полный каталог STR 847 | В разработке — сейчас 24 демо |
